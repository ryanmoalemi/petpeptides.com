# Pet Peptides

Pet Peptides website - Information about peptides for pet health and wellness.

## Setup

This site is hosted on GitHub Pages at petpeptides.com.

### Local Development
- Simply open `index.html` in a browser for local testing
- All content is static HTML/CSS/JavaScript

### Deployment
- Push changes to the main branch
- GitHub Pages automatically deploys the site

## File Structure
- `index.html` - Main landing page
- `CNAME` - Custom domain configuration for GitHub Pages
- `css/` - Stylesheets
- `js/` - JavaScript files
- `images/` - Image assets
